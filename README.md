# Lodeur Unique - Perfumaria

## O que é ?
É um site que simula um e-commerce de uma perfumaria, a Lodeur Unique. Nesta aplicação, pode-se observar a página inicial, catálogo, os produtos, carrinho e até mesmo um formulário simples de contato com a empresa. 

## Objetivos
Treinar técnicas e boas práticas de:
- Frontend responsivo, intuitivo e moderno
- Versionamento de códigos
- Colaboração com uma equipe real

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript

## Como Executar

1. Clone o Repositório:
```bash
git clone https://github.com/damasceno635/Lodeur_Unique.git
```

2. Entre na pasta:
```bash
cd Lodeur_Unique
```

3. Baixe a extensão "Live Server"

4. Clique com botão direito no index.html e depois em “Open with Live Server”
